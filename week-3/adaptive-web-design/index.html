Adaptive web design (AWD)

Adaptive web design (AWD): it is a web approach where website uses multiple fixed approach designed for specific screen sizes or desktop types e.g phones, tablet,laptops and so on. The site detects user's device or screen width and serves the most appropraite layouts.


Differences between Adaptive-web-design and Responsive-web-design

The key difference between Adaptive-web-design and Responsive-web-design is RWD uses a single flexible layout that adjust fluidly to any screen size using css especially media queries, while, AWD uses multiple fixed layouts each designed for specific screen sizes and swiches betwwen them based on the device type or screen width


Feature Responsive-web-design Adaptive-web-design
Layouts Fluid and flexibility Fixed at breakpoints
Adjust using: Css media queries Device detection and layouts switch
Number of layouts One flexible layout Multiple flexible layouts
Device specific: No Yes
Development effort Uually simpler Can be more complex
