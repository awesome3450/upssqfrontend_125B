Css-fonts

In css, we use the font-family property to specify the font of the text.

Note:If the font name is more than one word , it must be in quotation mark. like: 'Times New Roman'

Tip:The font family property should hold several fonts names as a fall back system. The browser will try to use te first one in the list, if not available, it will use the next one.

Example

.p1{
font-family: 'Times New Roman', Times, serif;
}
.p2{
font-family: Arial,Helvetica,sans-serif;
}
.p3{
font-family: "lucida Console", "Courier New", monospace;
}
