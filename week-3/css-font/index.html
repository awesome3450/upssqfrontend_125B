Font selection

The right font can create the strong identity for your brand.


Using a font that is easy to read is very important. The font adds value to your text. It is also important to choose the correct color and text size for the font.

Generic font families

In css, there are five generic font families:


  1. Font family-serif: Serif font has a small stroke at the edges of each letter, they create a sense of formality and elegance.
  2. Sans-serif: These fonts have clean lines(no small strokes attached). They create a mordern and minimalistic look.
  3. Mono-space: These fonts have all their letters with the same fixed width , they create a mechanical look.
  4. Cursive fonts: These fonts immitate human handwriting.
  5. Fantasy: These are decorative/playful fonts.

All the different font names belong to one of the generic family. In design, font-family are referred to as type phase.
