Css media query

The @media rule introduced in css2 made it possible to define diffrent style rules for different media types.

Medua queries in css3 extended the css2 media type idea: instead of looking for a type of device , they look at the capability of the device

Medai queries can be used to check many things such as:

  • Width and height of the viewport
  • Width and height of the device
  • Orientation of the viewport(landscape or portrait)
  • Orientation of the device(landscape or portrait)
  • Resolution of the device

Using media queries is a popular technique for delivering a tailored style sheet to desktops, laptops, tablets, and mobile phones such as; iphone and android.
