Responsive-web-design(RWD)

Responsive web design (RWD) is a design approach that ensures a website looks and function well across a wide range of devices and screen sizes -from mobile phones to tablets to desktop computers-without requiring separate versions of the site.

Key concepts of responsive web design

  1. Fluid grids:

    Layouts are built using relative units like percentages instead of using fixed units like pixels allowing elements to resize based on the screen.

  2. Flexible images:

    Images scale within their containing elements so they don't overflow or distort on different screen sizes.

  3. Media query:

    Css rules that apply styles based on the device characteristic .

    Example:
    

                        
    
                            @media(max-width:768px){
                                  .menu{
                                    display: none;
                                  }
                            }

    Mobile-file approach

    Design start with the smallest screen size and scales up ensuring a good user experience and mobile devices first.
