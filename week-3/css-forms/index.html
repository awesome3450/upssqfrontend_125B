The form element

The html form element is used to create an html form for user input.

The form element is a container for different type of input elements such as text fills, check boxes, radio buttons, and submit buttons.

Input element

The html input element is the most used element

Type Description
Input type text It displays the single line text input fill
Input type radio It defines a radio button input fill that allows selection of one option from a set.
Checkbox It defines a checkbox page that allows selection of multiple options from a set.
Input type number It defines an input fill from entering a number.
Input type email It defines an input fill for entering an email address.
Input type password It defines a password input fill.
