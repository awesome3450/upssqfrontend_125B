Te float property

The float property is used for positioning and formatting contents e.g when u want an image to float left in the container.

The float propety can have one of the following values:

  • left- the elements flows to the left of the container
  • right- the elements flows to the right of the container
  • none- the elements does not float, it is the default property.
  • Inherit- the elements inherit the float value of its parents.

In its simplest use the float property is used to wrap text around images.
