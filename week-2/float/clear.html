The clear property

When we use the floating property, and we want te next element to be below

The clear property specifies what should happen to the element that is next to the floating element.

The clear property can have one of the following values:

    the element is not pushed beow left or right element
  • None- The elements
  • Left- the element is pushed below left floated element
  • Right- the eleent is pushed below right floated element.
  • Both- the element is puhed below right and left floated element.
  • Inherit- the element inherit the clear value from the parent.
