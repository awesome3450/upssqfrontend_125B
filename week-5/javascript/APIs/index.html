Browsers APIs

Browsers APIs are built into your web browser and are able to expose data from the surrounding computer environment or do useful complex things.

  • Document Object Model APIs

    The document objecty model(DOM) helps u to manipulate html and css by creating, removing and changing html by dynamically applying new style to ur page among other things. Anytime u see a popup window appear on a page or some new contents displayed , that is The DOM action like in the example above.

  • The Geo Location APIs

    The Geo Location APIs retrieves geographical information. This is how Google Maps is able to find ur location and plot it on a map.

  • The Canvas & WebGL APIs

    They allow u to create animated 2D and 3D graphics.

  • Audio & Video APIs

    Example: Html Media elements and WebRTC allows u to do many interesting things with multi-media that is, play audio and video right in the webpage or grab video from ur web camera and display it on someone's else computer.
