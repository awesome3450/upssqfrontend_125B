A high level definition

Javascript is a scripten or programming language that allows you to implement complex features on web pages. Anytime a web page does more than just sit there to display statics information, popups, it is most ikely that Javascript is involved. Javascript is a third layer of the three standard technologies, the others being html and css.

  • Html

    Html is the markup language that we use to structure and give meaning to the web content e.g defining paragraphs, headings and data tables for embedding videos and images in a page.

  • Css

    Css is the language of style rules that we use to apply styling to our html content e.g setting background colors and fonts.

  • Javascript

    Javascript is a scripting content that enable you to create dynamically updating content, controlling multi-media, animate images and pretty more everything else.

Browser Security

Each browser tab has its own separate bucket for running code(This buckets are called execution environment in technical terms). This means that in most cases, the code in each tab is run completely separately and the code in one tab can't directly affect the code in another tab.

Interpreted VS Complied code

You might hear the terms, interpreted and complied in the context of programming. In intetrpreted languages, the code is run from top to bottom and the results of running the code is immediately returned. You don't have to transform the code into a different form before the browser runs it, the code is received in the programmer friendly text form and processed directly from that.


Complied language on the other hand, are transformed or complied into another form before they are run on the computer e.g C/C++ are complied into machine code that is then run by the computer. The program is executed from a binary format/ byte-code, which was generated from the original source code.


Javascript is a light weight interpreted programming language. The web browser recieves the Javascript code in its original text form and runs the script from that. From a technical stand point, most modern javascript interpreters actually use a technique called Just-In -Time compiling(JIT) to improve performance; the javascript source code gets complied into a faster binary format while the script is being used so tht it can be run as quickly as possible. However, javascript is still considered an interpreted language since the compliation is handled at run-time rather than ahead of time.
